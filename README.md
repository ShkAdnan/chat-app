# Chat APP using VUE + Jetstream + Laravel

![Screenshot](screenshot.PNG)

> Laravel 8.50 

## Quick Start

``` bash
# Install Dependencies
composer install
npm install

# Run Migrations
php artisan migrate

# Add virtual host if using Apache

# If you get an error about an encryption key
php artisan key:generate


# Run Application

```
